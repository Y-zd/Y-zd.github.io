什么是 Hexo？

  • Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
  • Hexo 拥有活跃的社区，有众多可用的开源主题可用（如本人使用的hexo-theme-butterfly主题）

安装教程（windows环境）

  • 需要的环境
    1. 安装git(安装git是为了将项目部署到github或gitee)
    2. 安装Node.js
      1. 官方下载安装
      2. 安装完成后控制台输入node -v查看是否安装成功
      3. 由于网络原因可替换 npm 的安装源为淘宝镜像
  • 开始hexo使用(官方文档)
    1. 安装hexo
    2. 创建根项目根目录进行项目初始化
    3. 项目初始化完成后会在根目录下生成以下结构
    4. 常用命令(所有命令支持hexo 首字母 形式)

主题

  • 可从网上寻找自己喜欢的主题文件放在themes文件下并将根目录下_config.yml中theme参数设置为对应主题
  • 如本人使用的hexo-theme-butterfly主题
    1. 上面的官方的配置文档很详细，照着配置就ok了
    2. 对配置评论功能，个人建议使用Gitalk这种不需要第三方小众网站支持的方式。但Gitalk调用github认证接口时会存在被墙，且网络也不稳定
    3. 使用Gitalk配置评论功能，设置OAuth Apps时，需要注意以下：
      1. Homepage URL 需要填写如 https://y-zd.github.io/ 个人使用gitpages的网址
      2. Authorization callback URL 需要填写如 https://blog.yanzhidong.com/ 使用gitpages配置的个人域名
      3. 在butterfly.yml 配置repo时，只需要填写如 Y-zd.github.io 个人github上的项目名，项目可以是自己任意的项目，评论原理为在所填项目的issues下评论
